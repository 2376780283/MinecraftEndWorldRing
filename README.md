如圖所示，紫色區域即為外島在絕對邊界內可生成的範圍。

記常數C=43748181634

設第n個圓形邊緣（不分環內外邊）半徑為rₙ，則由圖像表達式可得：rₙ=√(Cnπ)，n∈*N

圓形區域的面積為：

S₀=πr₁²＝Cπ²

第n個環形區域的面積為：

Sₙ=πr₂ₙ₊₁²-πr₂ₙ²=π【Cπ(2n+1)-Cπ(2n)】=Cπ²

即S₀=S₁=S₂=…=Sₙ，每個區域的面積相等。

补充一下这个常数的精确值:2^37/π
